将具有AI特征信息的矩阵加密至图片

整体过程

将具有AI特征信息的矩阵加密至图片总共包含：矩阵信息处理、rsa加密、水印信息嵌入三个步骤，提取执行逆过程即可，具体图示如下：

矩阵信息的处理

提取出的AI特征信息的矩阵非常大，如果直接嵌入到图片中会导致水印信息过长，无法全部嵌入，因此我们采用了….

RSA加密

1. 概述

本项目实现了使用 RSA 算法对矩阵进行加密和解密的功能。矩阵首先被转换为字符串，然后通过 RSA 算法加密。加密后的内容以十六进制形式输出，并存储在字符串中。解密时，程序从存储的密钥中读取私钥，对加密的字符串进行解密并还原为原始的矩阵字符串。

2. 加密流程

函数：encryptMatrix_RSA(matrix)

该函数将输入矩阵加密并返回加密后的字符串。具体步骤如下：

  1. 矩阵转换：使用 mat2str(matrix) 将输入的矩阵转换为字符串形式。
  2. 密钥生成： 随机生成 RSA 公钥和私钥，调用 createKeys 函数生成 n, e, d 三个参数，其中 n 是模数，e 是公钥，d 是私钥。
  3. 密钥存储：生成的公钥和私钥分别存储在 /key/public.txt 和 /key/private.txt 文件中，调用 storeKeys 函数完成存储。
  4. RSA 加密：遍历矩阵字符串的每个字符，将其转换为 ASCII 码后使用 fastExpMod 函数进行 RSA 加密，并将加密结果转换为十六进制字符串。

3. 解密流程

函数：decryptMatrix_RSA(encryptedString)

该函数用于解密加密后的字符串并返回原始的矩阵字符串。具体步骤如下：

  1. 读取私钥：从 /key/private.txt 中读取存储的私钥 n 和 d，使用 loadPrivateKey 函数完成。
  2. RSA 解密：按照加密时的分段方式，解码十六进制的加密字符块，并通过 fastExpMod 函数进行 RSA 解密，将解密后的 ASCII 值转换为字符，拼接得到完整的字符串。

4. 核心算法说明

4.1 快速模幂算法 (fastExpMod)

此函数用于计算 b^e mod m，使用了快速幂算法，保证了在大整数的加密和解密过程中计算的高效性。

4.2 密钥生成 (createKeys)

该函数生成两个不同的 64 位素数 p 和 q，并通过 n = p * q 和欧拉函数 φ(n) = (p-1)*(q-1) 计算出模数 n 和公钥、私钥指数 e 和 d。素数的生成使用了 Miller-Rabin 素性测试算法来保证生成数的正确性。

5. 密钥存储与读取

5.1 存储密钥 (storeKeys)

该函数将生成的公钥和私钥分别存储在 /key/public.txt 和 /key/private.txt 文件中，供加密和解密使用。

5.2 读取私钥 (loadPrivateKey)

该函数从私钥文件 /key/private.txt 中读取私钥，返回存储的 n 和 d。

6. 文件结构

  • encryptMatrix_RSA.m: 执行矩阵的加密操作。
  • decryptMatrix_RSA.m: 执行解密操作。
  • /key/public.txt: 存储公钥的文件。
  • /key/private.txt: 存储私钥的文件。

基于DCT变换的水印信息的嵌入

1. 图片信息的DCT变换

离散余弦变换（英语：discrete cosine transform, DCT），二维DCT变换十分适合图像处理，利用其“能量集中”的特性，将图像信息集中在少数低频系数上，而纹理和边缘信息则在中低频系数中，同时由于DCT作为一种常用的时域频域变换算法，对噪声的鲁棒性较好，图像经过DCT处理后，图像像素之间的相关性较低，信息主要集中于直流系数和几个低频系数中，其数字水印算法处理相对简单，另外根据人类视觉系统理论，人眼对图像平滑区域的变换比较敏感，对纹理区域的变换不太敏感，所以可以选择将水印信息存放至中、高频部分，保留低频系数，从而实现在最大化保留图片原本信息的基础上嵌入水印信息，实现为图片添加数字水印。

2. 具体嵌入思路

关于图片的处理：考虑到水印信息比较长以及DCT变换的基本需要，将图片进行4*4分块进行处理。

水印嵌入位置的选取：为了保证图片原始的信息以及加密过后图片的质量，以及考虑到压缩、外来攻击的影响等因素，选择4*4中3行3列（中低频），只选用一个位置进行嵌入的方法，同时选用一个较小的水印强度系数k。

嵌入及提取过程：首先对图片进行4*4分块分别进行DCT变换，将字符串转化为二进制ascii码形式，然后通过3/3位置的信息与k进行比较进行水印嵌入，对所有字符串执行以上过程，嵌入后执行逆DCT变换还原为原来的图片，提取过程直接进行逆操作即可。
